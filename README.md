# 10-LangGraph 🚀

## 🛠 초기 세팅 방법 (Initial Setup)

귀찮지만 한 번만 제대로 해주면 된다구!

### 1. 저장소 복제 (Clone Repository)

먼저 이 코드를 네 컴퓨터로 가져와!

```bash
git clone https://github.com/elixirevo/hk-langgraph-lecture.git
cd 10-langgraph
```

### 2. 파이썬 버전 고정 (Pin Python Version)

```bash
uv python pin 3.14
```

### 3. 의존성 설치 (Sync Dependencies)

이제 필요한 것들을 한 번에 설치할 시간이야!

```bash
uv sync
```
