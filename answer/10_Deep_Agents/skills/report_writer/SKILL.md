# Report Writer Skill

전문적인 보고서를 작성하는 스킬이에요.

## 보고서 유형

### 1. 비즈니스 보고서
경영진을 위한 간결하고 액션 중심적인 보고서예요:
- 핵심 요약 (Executive Summary)
- 주요 지표 및 KPI
- 권장 사항 및 다음 단계

### 2. 기술 문서
개발자를 위한 상세한 기술 문서예요:
- 시스템 아키텍처 설명
- API 레퍼런스
- 구현 가이드

### 3. 분석 리포트
데이터 기반의 인사이트 보고서예요:
- 배경 및 목적
- 방법론
- 분석 결과
- 결론 및 제언

## 작성 원칙

1. **명확성**: 독자가 이해하기 쉽게 작성해요
2. **구조화**: 논리적 흐름으로 정보를 배치해요
3. **시각화**: 데이터는 표나 차트로 제시해요
4. **실행 가능성**: 구체적인 권장 사항을 포함해요

## 응답 형식

보고서는 요청에 따라 Markdown 또는 구조화된 텍스트로 제공해요.

```markdown
# [보고서 제목]

**작성일**: [날짜]
**작성자**: AI 에이전트

---

## 요약 (Executive Summary)
[3-5문장으로 핵심 요약]

## 본문
[섹션별 상세 내용]

## 결론
[핵심 메시지와 권장 사항]
```

## 스타일 가이드

- 능동형 문장 사용
- 전문 용어는 처음 등장 시 설명
- 숫자는 구체적으로 제시 (예: "증가했다" → "15% 증가했다")
- 각 섹션은 3-7개 문장으로 구성
